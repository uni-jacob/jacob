    Добавление бота в чаты¶

    Приглашение¶

    На странице бота нажмите на кнопку {Добавить в беседу}, выберите нужные беседы из списка и нажмите {Пригласить}

    Бот автоматически поприветствует участников беседы.

    Затем нужно сообщить боту, как использовать чаты, зарегистрировав их в системе. См. Регистрация чатов
